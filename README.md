
## Setup project

```
git clone https://github.com/ooca-pratchaya/express-template.git
```

```
cd express-template/
```
## Remove `.git`

```
rm -rf .git/
```

## Install
```
yarn
```

lastly, rename folder to project name

## Project environment 
create `.env` file to root project and use copy those config to the file
```
ENV=1
```
